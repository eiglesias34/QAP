Autores:

Python 3

Archivos:
- AlgoritmoGenetico.py : implementación de algoritmo genetico
- AntColony.py : implementación de Colonia de Hormigas
- ReadData.py : permite leer archivos de datos y resultados
- QAP.py : clase QAP
- QAP_Poblacion.py : clase QAP modificada para trabajar con poblaciones
- LocalSearch.py : implementacion de busqueda local
- main.py : main del proyecto

Para ejecutar en Linux: 
> python main.py inputFile solFile 

Ejemplo: 
> python3 main.py qapdata/tai64c.dat qapsoln/tai64c.sln
